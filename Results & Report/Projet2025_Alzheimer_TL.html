Question 2.a) Pourquoi estimer \(\beta\) et pas \(\lambda_{nc}\)

La cohorte est seulement constituée de familles recrutées à partir d’un cas index atteint jeune, et porteur d’un variant SORL1.
Cela a introduit des biais de sélection (les âges de survenue de la maladie observés ne représentent pas ceux de la population générale).
Donc on ne peut pas estimer correctement \(\lambda_ {nc}\) à partir de ces données.
On peut cependant estimer \(\beta\) directement à partir de la cohorte familiale car \(\beta\) est spécifique à l’effet du variant SORL1, alors que le \(\lambda_{nc}\) est le risque de base qui peut être influencé par les biais.

Question 2.b) Estimation de beta

# Charger les paramètres nécessaires
load("APOE.mu.RData")
load("APOE.sigma.RData")
source("APOE.R")
# Numérateur : nombre de malades parmi les porteurs de SORL1
num = sum(dataEC[dataEC$SORL1.recode==1,]$status,na.rm=TRUE)

# Calcul du risque cumulé non causal (fonction fournie) pour chaque individu
dataEC$Lnc = mapply(Cumullambda.APOE,
                     nb4 = dataEC$APOE.nb4,
                     t   = dataEC$age,
                     u   = 0)

# Dénominateur : somme des risques chez les porteurs de SORL1
denom = sum(dataEC[dataEC$SORL1.recode == 1,]$Lnc, na.rm = TRUE)

# Estimation de β
beta = log(num / denom)
beta
[1] 3.407223

On va estimer \(\beta\) basée sur une vraisemblance conditionnelle comme dans le TP avec :
- en numérateur : le nombre de porteurs SORL1 ayant développé la maladie
- en dénominateur : la somme des risques cumulés non causals, calculés à l’âge observé pour chaque porteur, via une fonction Cumullambda.APOE().

Interprétation :
La valeur estimée de \(\hat{\beta} \approx 3.41\) indique que le risque de développer la maladie est multiplié par \(e^{3.41} \approx 30\) chez les porteurs du variant SORL1 a statut APOE égal.
Cette estimation est cohérente avec l’effet attendu d’un variant rare à fort effet.

Question 2.d) Explication IC

On a construit les intervalles de confiance à 95% :
- On avait déjà estimé \(\beta\) , avec la méthode ci dessus
- On va donc estimé l’erreur standard de \(\hat{\beta}\) qui, sous l’hypothèse d’indépendance est approximée par la racine inverse du dénominateur utilisé pour estimer \(\beta\)
- Ensuite :
En supposant que \(\hat{\beta}\) suit une loi normale, on construit l’intervalle :

\[ IC_{95\%} = \left[ \hat{\beta} - 1{,}96 \cdot SE(\hat{\beta}),\ \hat{\beta} + 1{,}96 \cdot SE(\hat{\beta}) \right] \]

-Finalement on applique à la courbe de pénétrance


On peut remarquer que pour SORL1 non porteurs : il n’y a pas de bande d’intervalles de confiance, ce qui est normal car le risque ne dépend pas de \(\beta\) .

Question 3.a) Estimer beta1, beta2, beta3

Dans ce modèle on suppose que l’effet du variant SORL1 dépend de l’âge.

\[ \beta(t) = \begin{cases} \beta_1 & \text{si } t < 70 \\ \beta_2 & \text{si } 70 \leq t < 80 \\ \beta_3 & \text{si } t \geq 80 \end{cases} \]

On a donc un taux de survenue de la maladie de :

\[ \lambda(t \mid a, s) = \lambda_{nc}(t \mid a) \cdot \exp(\beta(t) \cdot 1_{s = 1}) \]

avec :
- \(a\) désigne le génotype APOE
- \(s\) = 1 indique que l’individu est porteur d’un variant SORL1 d’intêret
- \(\lambda_{nc}(t \mid a)\) est le taux de base conditionnel à APOE

On passe à la log-vraisemblance conditionnelle

on note \(t_i\) l’âge d’observation (maladie ou censure) pour individu \(i\) et \(\delta_i\) = 1 si l’indiv a développé la maladie, 0 sinon.

\[ \log L(\beta) = \sum_i \left[ \log(S(t_i \mid a_i, s_i)) + \delta_i \cdot \log(\lambda(t_i \mid a_i, s_i)) \right] \]

Le taux de risque est donné par

\[ \lambda(t_i \mid a_i, s_i) = \lambda_{nc}(t_i \mid a_i) \cdot \exp(\beta(t_i) \cdot 1_{s_i = 1}) \]

donc ,

\[ \log(\lambda(t_i \mid a_i, s_i)) = \log(\lambda_{nc}(t_i \mid a_i)) + \beta(t_i) \cdot 1_{s_i = 1} \]

et la fonction de survie est

\[ S(t_i \mid a_i, s_i) = \exp\left( -\int_0^{t_i} \lambda(u \mid a_i, s_i) du \right) \]

On peut maintenant découper en tranche d’âges
on a 3 cas selon la valeur de \(t_i\)

si \(t_i\) < 70 :
\[ \log(S(t_i)) = -\Lambda_{nc}(t_i \mid a_i) \cdot \exp(\beta_1 \cdot 1_{s_i = 1}) \]

si 70 < \(t_i\) < 80 :

\[ \log(S(t_i)) = - \Lambda_{nc}(70 \mid a_i) \cdot \exp(\beta_1 \cdot 1_{s_i = 1}) - \left( \Lambda_{nc}(t_i \mid a_i) - \Lambda_{nc}(70 \mid a_i) \right) \cdot \exp(\beta_2 \cdot 1_{s_i = 1}) \]

si \(t_i\) > 80 :

\[ \log(S(t_i)) = - \Lambda_{nc}(70 \mid a_i) \cdot \exp(\beta_1 \cdot 1_{s_i = 1}) - \left( \Lambda_{nc}(80 \mid a_i) - \Lambda_{nc}(70 \mid a_i) \right) \cdot \exp(\beta_2 \cdot 1_{s_i = 1}) - \left( \Lambda_{nc}(t_i \mid a_i) - \Lambda_{nc}(80 \mid a_i) \right) \cdot \exp(\beta_3 \cdot 1_{s_i = 1}) \]

On va calculer les estimateurs en maximisant la log-vraisemblance logL( $ \beta$ ) , donc on annule les dérivées partielles :

\[ \frac{\partial \log L}{\partial \beta_j}(\hat{\beta}) = 0 \quad \text{ pour } j = 1, 2, 3 \]

donc pour les indiv atteints avant 70 ans :

on avait

\[ \log L_1(\beta_1) = \sum_{t_i < 70,\ s_i = 1} \left[ - \Lambda_{nc}(t_i \mid a_i) \cdot \exp(\beta_1) + \delta_i \cdot \beta_1 \right] \]

on dérive en \(\beta_1\)

\[ \frac{\partial \log L_1}{\partial \beta_1} = \sum_{t_i < 70,\ s_i = 1} \left[ - \Lambda_{nc}(t_i \mid a_i) \cdot \exp(\beta_1) + \delta_i \right] \]

on cherche le point où cette dérivée est nulle

\[ \sum_{t_i < 70,\ s_i = 1} \left[ - \Lambda_{nc}(t_i \mid a_i) \cdot \exp(\hat{\beta}_1) + \delta_i \right] = 0 \]

\[ \exp(\hat{\beta}_1) = \frac{ \sum_{t_i < 70,\ s_i = 1} \delta_i }{ \sum_{t_i < 70,\ s_i = 1} \Lambda_{nc}(t_i \mid a_i) } \]

\[ \hat{\beta}_1 = \log \left( \frac{ \sum_{t_i < 70,\ s_i = 1} \delta_i }{ \sum_{t_i < 70,\ s_i = 1} \Lambda_{nc}(t_i \mid a_i) } \right) \]

En appliquant la même méthode pour \(\beta_2\) et \(\beta_3\) on trouve :

pour les indiv atteints entre 70 et 80 ans :

\[ \hat{\beta}_2 = \log \left( \frac{ \sum_{70 \leq t_i < 80,\ s_i = 1} \delta_i }{ \sum_{70 \leq t_i < 80,\ s_i = 1} \left[ \Lambda_{nc}(t_i \mid a_i) - \Lambda_{nc}(70 \mid a_i) \right] } \right) \]

pour les indiv atteints après 80 ans :

\[ \hat{\beta}_3 = \log \left( \frac{ \sum_{t_i \geq 80,\ s_i = 1} \delta_i }{ \sum_{t_i \geq 80,\ s_i = 1} \left[ \Lambda_{nc}(t_i \mid a_i) - \Lambda_{nc}(80 \mid a_i) \right] } \right) \]

Ces estimateurs quantifient l’effet du variant SORL1 dans trois tranches d’âge distinctes.
En séparant l’âge en intervalles, on peut observer si l’effet du variant est plus marqué à certains âges.

nb_iteration = 100  # Nombre d'itérations
age = 40:100  # Plage d'âges

# Fonction pour calculer la pénétrance en fonction de l'âge, du nb4 et de beta
calculate_penetrance <- function(age, nb4, beta) {
  L = Cumullambda.APOE(nb4 = nb4, t = age, u = 0)  # Utilisation de la fonction Cumullambda
  return(1 - exp(-L * exp(beta)))
}

# Initialisation des matrices de pénétrance complexe
matrice_penetrance0_complexe = matrix(NA, nrow = length(age), ncol = nb_iteration)
matrice_penetrance1_complexe = matrix(NA, nrow = length(age), ncol = nb_iteration)
matrice_penetrance2_complexe = matrix(NA, nrow = length(age), ncol = nb_iteration)

# Calcul des pénétrances et remplissage des matrices
for (i in 1:length(age)) {
  age_val = age[i]
  for (j in 1:nb_iteration) {
    nb4 = sample(0:2, 1, replace = TRUE)  # Exemple : génération aléatoire du nombre d'allèles 4
    beta = sample(c(beta1, beta2, beta3), 1)  # Exemple : génération aléatoire d'un beta parmi les trois
    penetrance = calculate_penetrance(age_val, nb4, beta)  # Calcul de la pénétrance en fonction des paramètres
    if (nb4 == 0) {
      matrice_penetrance0_complexe[i, j] = penetrance
    } else if (nb4 == 1) {
      matrice_penetrance1_complexe[i, j] = penetrance
    } else {
      matrice_penetrance2_complexe[i, j] = penetrance
    }
  }
}

# Fonction pour tracer les courbes de pénétrance en fonction de beta1, beta2, et beta3 et inclure les intervalles de confiance
plot_penetrance_avec_IC = function(beta1, beta2, beta3) {
  plot(age, type = "n", xlim = range(age), ylim = c(0, 1), xlab = "Âge", ylab = "Pénétrance", main = "Pénétrance, modèle complexe")
  
  # Calculer les intervalles de confiance pour chaque combinaison d'âge et de nb4
  for (nb4 in 0:2) {
    # Sélectionner la matrice de pénétrance appropriée
    if (nb4 == 0) penetrance_matrix = matrice_penetrance0_complexe
    else if (nb4 == 1) penetrance_matrix = matrice_penetrance1_complexe
    else penetrance_matrix = matrice_penetrance2_complexe

    # Calculer la moyenne des pénétrances pour chaque âge
    mean_penetrance = apply(penetrance_matrix, 1, mean, na.rm = TRUE)
    
    # Calculer les intervalles de confiance pour chaque courbe de pénétrance
    IC_borne_inf = apply(penetrance_matrix, 1, quantile, probs = 0.025, na.rm = TRUE)
    IC_borne_sup = apply(penetrance_matrix, 1, quantile, probs = 0.975, na.rm = TRUE)
    
    # Tracer la courbe de pénétrance
    lines(age, mean_penetrance, col = c("red", "orange", "blue")[nb4 + 1], lwd = 2)
    
    # Tracer l'intervalle de confiance
    lines(age, IC_borne_inf, col = c("red", "orange", "blue")[nb4 + 1], lty = 2)
    lines(age, IC_borne_sup, col = c("red", "orange", "blue")[nb4 + 1], lty = 2)
  }
  
  # Légende
  legend("topleft", fill = c("red", "orange", "blue"), border = NA, lty = c(1, 1, 1), legend = c("0 allèle 4", "1 allèle 4", "2 allèles 4"))
}

construc_famille=function(){
  #génération des identifiants des individus
  famid=rep(1,5)
  indid=c(1,2,3,4,5)
  sexe=sample(c(1,2),5,replace=TRUE)
  patid=sample(c(0,1),5,replace=TRUE)
  matid=sample(c(0,2),5,replace=TRUE)
  
  #détermination du génétype pour chaque individu
  variant=rep(0,5)
  parent_porteur=sample(1:2,1) #choix aléatoire du parent porteur
  variant[parent_porteur]=1 #le parent porteur transmet le variant
  
  #transfert du variant aux enfants
  for (i in 3:5){
    if (variant[indid[i]]==0){
      variant[indid[i]]=sample(0:1,1)
    }
  }
  
  #génération aléatoire de l'âge de début de la maladie
  debut=-log(runif(5))/theta
  debut[variant==0]=Inf #les non porteurs ne développent pas la maladie
  
  #génération aléatoire de l'âge de censure
  censure=rnorm(5,mean=65,sd=15)
  
  #création des colonnes statut et age
  statut=ifelse(!is.infinite(debut),1,0) #seul les individu ayant un age de début de maladie peuvent avoir l'étiquette 1 pour statut
  age=ifelse(!is.infinite(debut),debut, censure)

  
  #retourner les données de la famille sous forme de dataframe
  data.frame(famid,indid,sexe,patid,matid,variant,statut,age)
}

#fonction pour sélectionner les familles avecu un cas index

}
